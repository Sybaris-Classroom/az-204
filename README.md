# AZ-204 Training Lab

Ce repository contient les **supports pratiques d'une formation Azure Developer Associate (AZ-204)** que j'ai concu.

Vous y trouverez :
- des **travaux pratiques (TP)** pour manipuler Azure à réaliser par les participants
- des **démonstrations guidées** faites par le formateur, mais réalisables aussi par les participants disposant de leur propre abonnement Azure
- des **quiz** pour tester votre compréhension des concepts clés

L'objectif est de vous aider à **comprendre concrètement les services Azure utilisés dans l'examen AZ-204**, et à vous entraîner avec des exercices proches des scénarios réels.

# TP vs Démos

Les **démos** sont rédigées sur le même principe que les TP : sous forme de tutoriels pas à pas, faciles à suivre et à reproduire.

La différence principale est que les démos sont réalisées par le formateur. Elles peuvent nécessiter des **plans tarifaires Azure** non accessibles dans l’environnement fourni aux étudiants pendant la formation.

Cependant, si vous disposez de votre propre abonnement Azure, vous pouvez tout à fait réaliser les démos comme des TP. C’est d’ailleurs recommandé si vous suivez la formation en autonomie.

Autre distinction importante : les **TP s’inscrivent dans un fil rouge**, avec la construction progressive d’une application réutilisée d’une séance à l’autre. Les démos, quant à elles, sont indépendantes et n’enrichissent pas cette application.
